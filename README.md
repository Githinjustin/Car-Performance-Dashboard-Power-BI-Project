# Car-Performance-Dashboard-Power-BI-Project
This project is an interactive Car Performance Dashboard built to analyze and compare high-performance vehicles across multiple brands. It provides a visually rich and dynamic interface to explore key automotive metrics such as price, horsepower, torque, and top speed.                                                                                           
🎯 Objectives                                                                                                               
Build a visually appealing and interactive dashboard                                                                         
Analyze and compare car performance metrics                                                                                  
Enable dynamic filtering for better exploration                                                                              
Present complex data in a simple, intuitive way 

📊 Key Features 

🔍 Interactive Filters (Slicers)                                                                                            
- Company / Brand selection                                                                                                  
- Car model selection                                                                                                        
- Fuel type filtering                                                                                                        
- Engine type filtering
-                                                                                                 
📈 KPI Cards   

💰 Average Price                                                                                                            
⚡ Horsepower (HP)                                                                                                          
🔧 Torque (Nm)                                                                                                              
🚀 Top Speed (km/h)    

🚘 Dynamic Car Visualization    

- Displays selected car image based on filters                                                                               
- Enhances user experience with real-time updates                                                                             
🏎️ Brand Navigation                                                                                                          
- Clickable brand icons (Audi, BMW, Bugatti, Ferrari, Lamborghini, Rolls-Royce, Volkswagen)                                  
- Smooth switching between different manufacturers
  
🎨 Clean UI/UX Design

- Minimalist and modern layout                                                                                               
- Balanced use of visuals and data                                                                                           
- Focus on readability and interaction
  
🛠️ Tools & Technologies

- Power BI                                                                                                                   
- Data Modeling                                                                                                              
- Data Cleaning & Transformation

🚀 How to Use

Open the .pbix file in Power BI                                                                                              
Use slicers to filter by brand, model, or specifications                                                                     
Click brand icons to navigate between manufacturers                                                                          
Analyze KPIs and compare performance instantly                                                                               

📈 Insights You Can Get

Compare performance across different car brands                                                                              
Identify high-performance vehicles                                                                                           
Analyze how specifications vary by model                                                                                     
Explore relationships between speed, power, and price
